Detailed Description

The message digest virtualnterface class.

This class can be used to create functions that will accept any of the implemented message digest instances as a parameter.

◆ ParallelMaxDegree()

virtual void IDigest::ParallelMaxDegree ( size_t  Degree)
pure virtual

Set the number of threads allocated when using multi-threaded tree hashing processing.

Thread count must be an even number, and not exceed the number of processor cores. Changing this value from the default (8 threads), will change the output hash value.

Parameters
DegreeThe number of threads to allocate
Exceptions
CryptoCipherModeExceptionThrown if the degree parameter is invalid

Implemented in Blake256, Blake512, SHA512, Skein256, Skein512, SHA256, Skein1024, Keccak1024, Keccak512, and Keccak256.

◆ ParallelProfile()

virtual ParallelOptions& IDigest::ParallelProfile ( )
pure virtual

Read/Write: Parallel and SIMD capability flags and sizes

The maximum number of threads allocated when using multi-threaded processing can be set with the ParallelMaxDegree(size_t) function. The ParallelBlockSize() property is auto-calculated, but can be changed; the value must be evenly divisible by the profiles ParallelMinimumSize() property. Changes to these values must be made before the Initialize(SymmetricKey) function is called.

Implemented in Blake256, Blake512, SHA512, Skein256, Skein512, SHA256, Skein1024, Keccak1024, Keccak512, and Keccak256.
