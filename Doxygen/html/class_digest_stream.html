Detailed Description

Digest stream helper class.

Wraps Message Digest stream functions in an easy to use interface.

Example of hashing a stream:

SHA512* eng = new SHA512();
StreamDigest sdgt(eng);
// get the hash code
hash = sdgt.Compute(Input);
delete eng;

Implementation Notes:

  • Uses any of the implemented Digests using either the IDigest interface, or a Digests enumeration member.
  • Implementation has a Progress counter that returns total sum of bytes processed per either of the Compute() calls.

◆ ParallelProfile()

ParallelOptions & DigestStream::ParallelProfile ( )

Read/Write: Contains parallel settings and SIMD capability flags in a ParallelOptions structure.

The maximum number of threads allocated when using multi-threaded processing can be set with the ParallelMaxDegree(size_t) function. The ParallelBlockSize() property is auto-calculated, but can be changed; the value must be evenly divisible by the profiles ParallelMinimumSize() property. Note: The ParallelMaxDegree property can not be changed through this interface, use the ParallelMaxDegree(size_t) function to change the thread count and reinitialize the state.
