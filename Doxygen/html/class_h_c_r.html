Detailed Description

An implementation of an HMAC Counter mode PRNG.

Uses a keyed SHA2 HMAC DRBG run in counter mode to generate pseudo-random output..

Example of generating a pseudo-random integer:

HCR rnd([SHA2Digests], [Providers], [Buffer Size]);
int num = rnd.NextUInt32([Minimum], [Maximum]);

Implementation Notes:

  • Wraps the Digest Counter mode Generator (HCG) DRBG implementation.
  • Can be initialized with any of the implemented SHA2 digests.
  • Uses an internal entropy provider to seed the underlying DRBG.

Guiding Publications:

  1. NIST SP800-90A: Appendix E1.
  2. NIST SP800-90B: Recommendation for the Entropy Sources Used for Random Bit Generation.
  3. NIST Fips 140-2: Security Requirments For Cryptographic Modules.
  4. NIST SP800-22 1a: A Statistical Test Suite for Random and Pseudorandom Number Generators for Cryptographic Applications.
  5. NIST Security Bounds for the NIST Codebook-based: Deterministic Random Bit Generator.
