explicit

Initialize with the default parameters.

Default settings are configured for sequential mode.

Parameters
OutputSizeDigest output byte length; set to 32 for Skein256, 64 for Skein512 or 128 for Skein1024
LeafSizeThe outer leaf length in bytes; this must be the digests block size
FanoutThe number of state leaf-nodes used by parallel processing (one state per processor core is recommended)

Parameters
NodeOffsetThe tree nodes relational offset
OutputSizeDigest output byte length; set to 32 for Skein256, 64 for Skein512 or 128 for Skein1024
VersionThe Skein version number; should always be a value of '1'
LeafSizeThe outer leaf length in bytes; this should be the digest block size in bytes
FanoutThe number of state leaf-nodes used by parallel processing (one state per processor core is recommended)
TreeDepthThe depth of the parallel tree; this value is always zero in this implementation
InfoOptional personalization string
