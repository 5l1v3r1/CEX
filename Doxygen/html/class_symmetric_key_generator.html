Detailed Description

A helper class for generating cryptographically strong keying material.

Generates an array, or an SymmetricKey or SymmetricSecureKey container class, using a definable cSHAKE(Provider+cutomization) generator.

Generate a symmetric key:

SymmetricKeyGenerator gen([SecurityPolicy], [Providers]);
// keysize with a 256 bit key and a 128 bit initialization vector
SymmetricKeySize ks(32, 16, 0);
// generate a symmetric key
SymmetricKey key = gen.GetKey(ks);

Implementation Notes:

  • Seed provider can be any of the Enumeration::Providers random providers.
  • The SecurityPolicy determines the expected cryptographic strength of the pseudo-random output (256. 512, or 1024)
