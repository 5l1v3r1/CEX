Detailed Description

A helper class for generating cryptographically strong keying material.

Generates an array, or an SymmetricKey or SymmetricSecureKey container class, using a definable Mac(Provider()) dual stage generator. The first stage of the generator gets seed material from the entropy provider, then Macs the seed and adds the result to the output key array.

Generate a symmetric key:

SymmetricKeyGenerator gen([Digests], [Providers]);
// keysize with a 256 bit key and a 128 bit initialization vector
SymmetricKeySize ks(32, 16, 0);
// generate a symmetric key
SymmetricKey key = gen.GetKey(ks);

Implementation Notes:

  • Seed provider can be any of the Enumeration::Providers generators.
  • Hash can be any of the Enumeration::Digests digests.
  • Default Prng is CSP, default digest is SHA512.
