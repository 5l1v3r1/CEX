◆ Initialize()

virtual void IStreamCipher::Initialize ( bool  Encryption,
ISymmetricKey &  Parameters 
)
pure virtual

Initialize the cipher with an ISymmetricKey key container.

If authentication is enabled, setting the Encryption parameter to false will decrypt and authenticate a ciphertext stream. Authentication on a decrypted stream can be performed by manually by comparing output with the the Finalize(Output, Offset, Length) function. If encryption and authentication are set to true, the MAC code can be appended to the ciphertext array using the Finalize(Output, Offset, Length) function.

Parameters
ParametersCipher key container. The LegalKeySizes property contains valid sizes
EncryptionUsing Encryption or Decryption mode
Exceptions
CryptoSymmetricCipherExceptionThrown if a null or invalid key is used

Implemented in ACS, ChaCha512, ChaCha256, Threefish512, Threefish256, and Threefish1024.

◆ ParallelMaxDegree()

virtual void IStreamCipher::ParallelMaxDegree ( size_t  Degree)
pure virtual

Set the maximum number of threads allocated when using multi-threaded processing.

When set to zero, thread count is set automatically. If set to 1, sets IsParallel() to false and runs in sequential mode. Thread count must be an even number, and not exceed the number of processor [virtual] cores.

Parameters
DegreeThe desired number of threads to allocate
Exceptions
CryptoSymmetricCipherExceptionThrown if an invalid degree setting is used

Implemented in ACS, ChaCha512, ChaCha256, Threefish512, Threefish256, and Threefish1024.

◆ ParallelProfile()

virtual ParallelOptions& IStreamCipher::ParallelProfile ( )
pure virtual

Read/Write: Parallel and SIMD capability flags and sizes

The maximum number of threads allocated when using multi-threaded processing can be set with the ParallelMaxDegree() property. The ParallelBlockSize() property is auto-calculated, but can be changed; the value must be evenly divisible by ParallelMinimumSize(). Changes to these values must be made before the Initialize(bool, ISymmetricKey) function is called.

Implemented in ACS, ChaCha512, ChaCha256, Threefish512, Threefish256, and Threefish1024.
