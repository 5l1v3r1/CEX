Detailed Description

An implementation of an entropy source provider using the system secure random generator

Example of getting a seed value:

std::vector<byte> output(32);
CSP gen;
gen.Generate(output);

On a windows system, the RNGCryptoServiceProvider CryptGenRandom() function is used to generate output. On Android, the arc4random() function is used. All other systems (Linux, Unix), use dev/random.

Guiding Publications::

  1. Microsoft RNGCryptoServiceProvider: class documentation.
  2. NIST SP800-90B: Recommendation for the Entropy Sources Used for Random Bit Generation.
  3. NIST Fips 140-2: Security Requirments For Cryptographic Modules.
  4. RFC 4086: Randomness Requirements for Security.
