◆ DistributionCode()

virtual std::vector<byte>& IDrbg::DistributionCode ( )
pure virtual

Read/Write: Reads or Sets the personalization string value in the KDF initialization parameters.

Must be set before Initialize(ISymmetricKey) is called. Changing this code will create a unique distribution of the generator. Code can be sized as either a zero byte array, or any length up to the DistributionCodeMax size. For best security, the distribution code should be random, secret, and equal in length to the DistributionCodeMax() size.

Implemented in HCG, BCG, and CSG.

◆ DistributionCodeMax()

virtual const size_t IDrbg::DistributionCodeMax ( )
pure virtual

Read Only: The maximum size of the distribution code in bytes.

The distribution code can be used as a secondary source of entropy (secret) in the KDF key expansion phase. For best security, the distribution code should be random, secret, and equal in size to this value.

Implemented in HCG, BCG, and CSG.
