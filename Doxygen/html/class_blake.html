Detailed Description

Contains the Blake2-256 and 512bit permutation functions.

The function names are in the format; Permute-rounds-bits-suffix, ex. PermuteR10P512C, 10 rounds, permutes 512 bits, using the compact form of the function.
The compact forms of the permutations have the suffix C, and are optimized for speed and low memory consumption (enabled in the hash function by adding the CEX_DIGEST_COMPACT to the CexConfig file).
The Unrolled forms are optimized for speed and timing-neutrality (suffix U), and the vertically vectorized functions have the V suffix.
The H suffix denotes functions that take an SIMD wrapper class (AVX2/AVX512) as the state values, and process input in SIMD parallel blocks.

This class contains horizontally vectorized (wide) forms of the functions; PermuteR10P8x512H and PermuteR12P4x1024H use AVX2, and PermuteR10P16x512H and PermuteR12P8x1024H use the AVX512 instructions.
These functions are not visible until run-time on some compiler platforms unless the compiler flag (AVX2 or AVX512) is explicitly declared.
