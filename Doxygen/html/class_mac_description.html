Detailed Description

The MacDescription structure.

Used in conjunction with the MacStream class. Contains all the necessary settings required to recreate a Mac instance.

Populating a MacDescription for an Hmac:

MacDescription md(
Digests.SHA512, // hmac engine
128); // key size in bytes
