◆ DomainKey()

virtual std::vector<byte>& IAsymmetricCipher::DomainKey ( )
pure virtual

Read/Write: Reads or Sets the Domain Key used by cSHAKE to generate the shared secret.

Changing this code will create a unique distribution of the cipher. The domain key can be used as a secondary secret shared between hosts in an authenticated domain. The key is used as a customization string to pre-initialize a custom SHAKE function, that conditions the SharedSecret in Encapsulation/Decapsulation. For best security, the key should be random, secret, and shared only between hosts within a secure domain.

Implemented in RingLWE, NTRU, McEliece, and ModuleLWE.
