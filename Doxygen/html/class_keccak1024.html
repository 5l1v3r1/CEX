Detailed Description

An implementation of the SHA-3 Keccak digest

Example using an IDigest interface:

Keccak1024 digest;
std:vector<byte> hash(digest.DigestSize(), 0);
// compute a hash
digest.Compute(Input, hash);

Implementation Notes:

  • The 1024bit version of Keccak is an extension, and should be considered as only for experimental use.
  • The 1024bit mode of Keccak uses a permutation with 48 rounds, twice the number of rounds as a standard Keccak trandform.
  • The output hash size is 128 bytes (1024 bits).
  • The input block size is 72 bytes (576 bits).
  • The Compute(byte[]) method wraps the Update(byte[], int, int) and Finalize methods./>
  • The Finalize(byte[], int) method resets the internal state.

Keccak1024::Keccak1024 ( KeccakParams &  Params)
explicit

Initialize the class with an KeccakParams structure.

The parameters structure allows for tuning of the internal configuration string, and changing the number of threads used by the parallel mechanism (FanOut). If the parallel degree is greater than 1, multi-threading hash engine is instantiated. The default thread count is 8, changing this value will produce a different output hash code.

Parameters
ParamsThe KeccakParams structure, containing the tree configuration settings.
Exceptions
CryptoDigestExceptionThrown if the KeccakParams structure contains invalid values

◆ ParallelProfile()

ParallelOptions & Keccak1024::ParallelProfile ( )
overridevirtual

Read/Write: Contains parallel settings and SIMD capability flags in a ParallelOptions structure.

The maximum number of threads allocated when using multi-threaded processing can be set with the ParallelMaxDegree(size_t) function. The ParallelBlockSize() property is auto-calculated, but can be changed; the value must be evenly divisible by the profiles ParallelMinimumSize() property. Note: The ParallelMaxDegree property can not be changed through this interface, use the ParallelMaxDegree(size_t) function to change the thread count and reinitialize the state, or initialize the digest using a KeccakParams with the FanOut property set to the desired number of threads.

Implements IDigest.
