Detailed Description

MAC stream helper class.

Wraps Message Authentication Code (MAC) stream functions in an easy to use interface.

Example of hashing a Stream:

SHA256* eng = new SHA256();
HMAC* mac = new HMAC(eng);
hmac->Initialize(Key, Iv);
MacStream ds(mac);
IByteStream* ms = new MemoryStream(Input);
Code = ds.Compute(ms);
delete eng;
delete mac;
delete ms;

Implementation Notes:

  • Uses any of the implemented Macs using the IMac interface.
  • Mac must be fully initialized before passed to the constructor.
  • Implementation has a Progress counter that returns total sum of bytes processed per either Compute() calls.
