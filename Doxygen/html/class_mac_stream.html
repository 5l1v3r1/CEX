Detailed Description

MAC stream helper class.

Wraps Message Authentication Code (MAC) function in an easy to use interface.

Example of hashing a Stream:

// instantiate the mac function
MacStream ms(Macs::HMACSHA256);
// initialize with a key
ms.Initialize(Key);
// wrap the input buyes in a memory stream
IByteStream* ms = new MemoryStream(Input);
Code = ds.Compute(ms);
delete ms;

Implementation Notes:

  • Uses any of the implemented Macs using the IMac interface, or a MAC enumeration type.
  • Implementation has a Progress counter that returns total sum of bytes processed per either Compute() calls.
