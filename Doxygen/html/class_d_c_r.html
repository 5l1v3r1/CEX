Detailed Description

An implementation of a Digest Counter PRNG.

Note* as of version 1.0.0.2, the order of the Minimum and Maximum parameters on the NextIntXX api has changed, it is now with the Maximum parameter first, ex. NextInt16(max, min).

Example of generating a pseudo random integer:

DCR rnd([Digests], [Providers], [Buffer Size]);
int num = rnd.NextUInt32([Minimum], [Maximum]);

Implementation Notes:

  • Wraps the Digest Counter mode Generator (DCG) drbg implementation.
  • Can be initialized with any of the implemented hash digests.
  • Can use either a random seed generator for initialization, or a user supplied Seed array.
  • Numbers generated with the same seed will produce the same random output.

Guiding Publications:

  1. NIST SP800-90A: Appendix E1.
  2. NIST SP800-90B: Recommendation for the Entropy Sources Used for Random Bit Generation.
  3. NIST Fips 140-2: Security Requirments For Cryptographic Modules.
  4. NIST SP800-22 1a: A Statistical Test Suite for Random and Pseudorandom Number Generators for Cryptographic Applications.
  5. NIST Security Bounds for the NIST Codebook-based: Deterministic Random Bit Generator.
